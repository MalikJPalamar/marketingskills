*April 2026 · Skill: paywall-upgrade-cro · Author: Malik Palamar*

---

## Paywall Contexts

Centaurion uses soft gates, not hard paywalls. The philosophy: give the framework freely, gate the depth and personalization.

| Gate | What's Behind It | What's Free | Phase |
|---|---|---|---|
| Assessment results email | Full Level analysis + roadmap | Level score | Phase 1 |
| Research report | Full PDF with all data | Key stats on web | Phase 2 |
| Course content | Video lessons + workbook | Glossary + methodology | Phase 2 |
| Certification | Certificate + directory listing | Course content | Phase 2 |

---

## Gate 1: Assessment Full Report (Phase 1)

### What's free (no gate)
- Level number (1–11)
- Level name and 2-sentence description
- "What separates this level from the next"

### What's gated (email required)
- Personalized gap analysis (specific to their answers)
- 3 highest-leverage actions for their level
- Recommended reading from the methodology

### Upgrade prompt copy
```
You're at Level 7: Background Agents

What separates Level 7 from Level 8 isn't a tool — it's an architecture decision.

Get your personalized Level 7 → 8 roadmap:
[Email address               ]
[Send my roadmap →          ]
```

**Optimization principles**:
- Show the level first (reciprocity — they got value before being asked)
- Specificity: "Level 7 → 8 roadmap" not "full report"
- No mentions of marketing emails, newsletters, or being "added to a list"

## Course Upgrade Flow (Phase 2)

### Free → Paid upgrade triggers

**Trigger 1: Assessment result → Course CTA**
Users who score Level 3–6 on the assessment are ideal course buyers (they're advanced enough to benefit, not so advanced they've self-solved).

```
[After showing Level 5 result]

At Level 5, you're MCP-connected. The jump to Level 8 (Multi-Agent Orchestration) 
is the highest-leverage move you can make in the next 90 days.

The Centaurion Method course walks you through it.

[See the course →]
```

**Trigger 2: Newsletter reader → Course CTA**
After 4+ newsletter issues, run a single upgrade email:

```
Subject: You've been reading for a month. Here's the next step.

[Name],

The newsletter gives you one signal per week.
The course gives you the full system.

If you're ready to go deeper:
→ centaurion.me/course

— Malik
```

### Pricing page upgrade CRO

- **Recommended tier highlighted** (Professional / Certification)
- **Decoy structure**: Starter makes Professional look like the obvious upgrade
- **Risk reversal prominent**: "14-day full refund, no questions asked"
- **One CTA per tier**: "Enroll in Starter" / "Get Certified" / "Team License"
- **No free trial**: Free content is the trial. The course is the commitment.

## Upgrade Psychology Principles for Centaurion

| Principle | Application |
|---|---|
| **Reciprocity first** | Framework is free. Assessment is free. Trust is built before any ask. |
| **Specificity** | "Level 7 → 8 roadmap" beats "full report" every time |
| **No artificial urgency** | Don't say "offer ends Friday" unless it actually does |
| **Transparent about what's behind the gate** | Tell them exactly what they'll get — the ICP is skeptical of vague promises |
| **Risk reversal** | 14-day refund removes the last objection for course enrollment |
